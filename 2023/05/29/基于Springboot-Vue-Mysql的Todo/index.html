虚拟机环境配置，开发生产环境应该全部在服务器端完成。

  1. java

使用vim的时候安装插件

后端项目在服务器上直接进行开发，练习vim等软件的使用。前端vue界面在本地实现后部署到服务器上。

采用前后端分离的方式进行开发。

Todo功能分析

登录功能。用户输入用户名密码进行验证完成登录。根据返回的结果显示登录信息。

注册功能。用户输入用户名和两次确认密码来完成注册功能。需要绑定邮箱号，发送邮件进行验证。

密码找回功能。用户输入邮箱后找回密码。

基础todo功能，一个todolisit。设定提醒时间（邮件提醒）。

准备使用vue，vue路由，vuex，以及elementuiplus来实现上面的需求。

微软界面如下，学习java开发首先模仿别人的项目开始做。
